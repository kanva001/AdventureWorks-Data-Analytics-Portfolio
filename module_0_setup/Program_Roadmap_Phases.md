# Program Roadmap  Phases
## AdventureWorks Inventory Accuracy & ICQA Analytics Program

The program is delivered in phased increments to ensure controlled scope expansion
and early value delivery. Each phase is executed through one or more Agile sprints
with formal release gates.

- Phase 0: Setup & Standards
- Phase 1: Excel ICQA Analytics
- Phase 2: SQL Analysis
- Phase 3: Data Quality & Validation
- Phase 4: Power BI Dashboards
- Phase 5: Capstone & Interview Readiness
